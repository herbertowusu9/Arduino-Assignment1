# Arduino Assignment 1 — Smart Countdown

This is my submission for Assignment 1 in Programming C++ for Engineers Using Arduino.

## What this program does

The program counts down from 5 to 1.

For each number:

- Prints the value to the Serial Monitor.
- Blinks the onboard LED that many times.

When finished, it prints:

=== Countdown Complete ===

## Concepts Used

- Variables
- Functions
- While Loops
- Serial Monitor
- LED Control
